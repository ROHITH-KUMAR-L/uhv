# Exty Scam Detector

A browser extension for real-time scam detection.

## Features
- **SSL Verification**: Heuristic check for secure connections.
- **URL Pattern Analysis**: Detects excessive subdomains and suspicious TLDs.
- **Lookalike Detection**: Flags domains mimicking popular brands.
- **Threatening Language Scanning**: Identifies high-pressure scam tactics.
- **Brand Integrity**: Cross-checks page branding with the actual domain.
- **Scam Lock Overlay**: Automatically blocks high-risk sites with a full-screen warning.
- **Real-time Live Scan**: A sidebar panel showing the analysis steps in real-time.
- **On-page Analysis Report**: Click the floating badge to see a full security breakdown.
- **Premium Glassmorphic UI**: High-end design aesthetics.

## Installation
1. Go to `brave://extensions/`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select this directory (`Exty`)
